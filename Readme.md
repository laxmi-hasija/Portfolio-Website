# 🌐 Full Stack Portfolio Website

A modern, responsive full-stack portfolio website built using **React**, **Node.js**, and **Express.js**. This website showcases my skills, projects, and experience while providing an easy way for recruiters and collaborators to connect with me. It also includes a downloadable resume and a contact form for seamless communication.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-github-username/PortfolioWebsite.git
cd PortfolioWebsite
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm run dev
```
